# /manual 現行UI同期パッチ

## 変更内容

- 旧CSSモックのホーム画面を、現行のホーム画面スクリーンショットへ変更。
- 旧CSSモックの連絡先A/B画面を、現行の「1人目／2人目」画面へ変更。
- 記録・履歴の案内を、現行の独立した「記録」タブへ変更。
- 現行アプリに存在しない1時間・3時間・6時間の通知一時停止案内を削除。
- OK成功はサーバー受付後に表示する説明へ同期。

## 適用先

`anpi-watch-landing` リポジトリ直下。

## 検証

```bash
node --test tests/manual-current-ui.test.mjs
npm run lint
npm run build
```
