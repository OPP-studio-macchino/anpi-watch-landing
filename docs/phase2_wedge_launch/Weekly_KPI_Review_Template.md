# Phase 2 Weekly KPI Review Template

## 結論

Phase 2では登録数を中心に見ない。NAD と 72h Armed Activation を中心に見る。

目的は、広く集めることではなく、高意図ユーザーが月額300円、2名URL同意、通知の限界を理解したうえで、72h Armedまで進むかを確認すること。

## Go / Pause / Stop 判断

### Go条件

- NADが増えている。
- 72h Armed Activation Rateが改善している。
- Invite-to-Consent CTRが改善している。
- Time to Second Consentが短くなっている。
- STOP率が悪化していない。
- Misleading Claims Inquiryが増えていない。
- 月額300円、2名URL同意、通知の限界が理解されている。

### Pause条件

- LP CTA clickはあるが、72h Armedに進まない。
- Contact A/B added から consent request sent で止まる。
- Time to Second Consentが長く、2名同意で詰まる。
- Price Objection Rateが上がる。
- Support Contact Rateが上がる。
- SMS failedが一時的に悪化する。

### Stop条件

- 救命保証の誤認が増える。
- 危険検知の誤認が増える。
- 通知到達保証の誤認が増える。
- 連絡先に過度な責任を負わせていると受け取られる。
- STOP率が継続的に上がる。
- L3+ Escalation Rate / 100 Armed Usersが悪化し、連絡先負担が増える。
- Broad Paid Social的な低意図流入が増える。

## ガードレール

- STOP率上昇。
- L3+率悪化。
- SMS失敗率悪化。
- 救命保証誤認。
- 料金誤認。
- 連絡先苦情。
- 2名同意前に通知が稼働すると誤解される。
- 通知が届いたように見えても、閲覧や対応を意味しないことが誤解される。
- 月額300円の説明が読まれていない。
