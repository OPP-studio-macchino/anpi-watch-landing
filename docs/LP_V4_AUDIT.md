# あんぴッチ LP v4 監査と改修方針

## 監査で確認した問題

- トップページが仕様説明の一覧になっており、初見ユーザーの「自分に必要か」を作れていなかった。
- 旧アイコン、夜景写真、汎用的なOKイメージが、現在のiOSアプリの藍色ブランドと一致していなかった。
- ヒーローからApp Storeまでの意思決定に、価値・利用条件・料金・限界が一本の流れで並んでいなかった。
- 家族・連絡先・本人の説明が同じ情報階層に混在していた。
- 48時間ルールは正確だったが、文章中心で理解コストが高かった。
- ヘッダーの項目数が多く、初見ユーザーに優先順位を委ねていた。

## 実施した改修

- ヒーローを「1日1回、OKを記録。途切れた時は、決めた人へ。」へ刷新。
- 実際のiOSスクリーンショットを使用し、架空UIを廃止。
- 連絡先画面と設定画面の個人情報を合成時に架空データへ置換。
- 問題提起 → 3ステップ → 実画面 → 48時間ルール → する/しない → 立場別導線 → 通知文 → 料金 → FAQ → 最終CTAへ再構成。
- ヘッダーを「仕組み / 通知の流れ / 家族・連絡先 / 料金 / FAQ」に整理。
- 新アイコン、favicon、Apple touch icon、OGP、Twitterカードを新ブランドへ統一。
- SoftwareApplication / FAQPageのJSON-LDを追加。
- モバイル固定CTA、スキップリンク、Reduced Motion対応を追加。

## 変更していない仕様

- 本人確認はSMSリンク方式。
- 連絡先は2名で、両方の明示同意が必要。
- 24 / 30 / 36 / 42 / 48時間の段階通知。
- STOPは即時有効。再開後も再同意が必要。
- 月額300円、通知ごとの追加料金なし。
- 危険・病気・事故・死亡・位置情報の検知、救命・発見・公的機関への自動通報は行わない。
